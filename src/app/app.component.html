Angular HTTP Example

We have already heard about string interpolation which is a technique to render a javascript string in html by using {{"{{ someText }}"}}.
This is pretty cool but we can even make a lot of string transformations inside the string interpolation by using pipes |
We can build custom pipes like a rot13 encoding pipe which takes a lowercase string and returns it returns an encrypted string.
Luckily we can append as many pipes as we want, therefore we can use the lowercase pipe before the rot13 pipe like that: {{"{{ secret | lowercase | rot13 }}"}}

Examples:

{{text}}

Lowercase

{{text | lowercase}}

Uppercase

{{text | uppercase}}

Euro

{{money | currency:"EUR"}}

Dollar

{{money | currency:"USD"}}

Date

Next christmas will be on {{christmas | date:"dd.MM.yyyy"}}

Date Uppercase

Next christmas will be on {{christmas | date | uppercase}}

Rot13 Encoded

{{secret | lowercase | rot13 }}

Rot13 Decoded

{{secret | lowercase | rot13 | rot13 }}
